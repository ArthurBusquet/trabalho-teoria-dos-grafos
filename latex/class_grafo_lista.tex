\doxysubsection{Detailed Description}
Implementação de um grafo usando listas encadeadas. 

Esta classe herda da classe abstrata {\ttfamily \doxylink{class_grafo}{Grafo}} e implementa suas funcionalidades utilizando listas encadeadas para armazenar os vértices e arestas.
