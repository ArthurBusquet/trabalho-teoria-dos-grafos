\doxysubsection{Detailed Description}
Implementação de um grafo utilizando matriz de adjacência. 

Esta classe herda da classe abstrata {\ttfamily \doxylink{class_grafo}{Grafo}} e implementa suas funcionalidades usando uma matriz 2D para armazenar as conexões entre os vértices.
